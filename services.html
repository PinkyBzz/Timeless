Pertanyaan Umum (FAQ)

Jawaban untuk pertanyaan yang sering diajukan.

Berapa lama proses pengerjaan project?
Waktu pengerjaan bervariasi tergantung kompleksitas. Untuk desain logo biasanya 3-7 hari kerja. Untuk fotografi dan videografi, proses editing memakan waktu 3-10 hari kerja setelah sesi pemotretan/syuting.
Apakah saya mendapatkan file mentah (RAW)?
Secara standar, kami memberikan file final yang sudah diedit (High Resolution JPEG/PNG/MP4). File mentah (RAW/Project File) dapat diberikan dengan kesepakatan khusus atau biaya tambahan.
Bagaimana sistem pembayarannya?
Kami menerapkan sistem DP (Down Payment) sebesar 50% di awal sebelum project dimulai, dan pelunasan 50% setelah project selesai dan file preview disetujui, sebelum pengiriman file final.
Apakah bisa request konsep custom?
Tentu saja! Kami sangat terbuka dengan ide-ide unik Anda. Silakan konsultasikan konsep yang Anda inginkan, dan kami akan membantu merealisasikannya.
